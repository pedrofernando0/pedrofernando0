# Pedro Fonseca

I build EdTech, structured-data workflows, and digital products for education teams that need clarity, traceability, and usable interfaces.

My work connects educational strategy, research operations, and software engineering: from auditable public-data maps to institutional systems, dashboards, and product prototypes.

## What I Build

- Learning and knowledge tools for educational institutions
- Structured data workflows with explicit sources, assumptions, and reproducible steps
- Web products using React, Next.js, TypeScript, JavaScript, Python, and SQL
- Product prototypes that turn complex institutional routines into maintainable systems
